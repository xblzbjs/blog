## 列表
列表分为无序列表和有序列表，无序列表用使用“-”、“+”、“*”任何一种都可以，有序列表使用数字加点“1.”的形式

### 无序列表
### Input
```
- 列表内容
+ 列表内容
* 列表内容
```
### Output
- 列表内容
+ 列表内容
* 列表内容

### 有序列表
“.”跟内容之间要有空格
#### Input
```
1. 列表内容
2. 列表内容
3. 列表内容
```
#### Output
1. 列表内容
2. 列表内容
3. 列表内容
