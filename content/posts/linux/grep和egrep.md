## 语法

### 第一种形式

```shell
grep [option] [pattern] [file, file2...]
```

### 第二种形式

```shell
command | grep [option] [pattern]
```

##  参数选项

### 必须掌握

| 选项 | 含义                                       |
| ---- | ------------------------------------------ |
| -v   | 不显示匹配航信息                           |
| -i   | 搜索时忽略大小写                           |
| -n   | 显示行号                                   |
| -r   | 递归搜索（只适合第二种语法）               |
| -E   | 支持扩展正则表达式                         |
| -F   | 不按正则表达式匹配，按照字符串字面意思匹配 |

### 了解即可

| 选项 | 含义                                       |
| ---- | ------------------------------------------ |
| -c   | 只输出匹配行的数量，不显示具体内容         |
| -w   | 匹配整词                                   |
| -x   | 匹配整行                                   |
| -l   | 只列出匹配的文件名，不显示具体匹配行的内容 |

## grep和egrep：

- grep默认不支持扩展正则表达式，只支持基础正则表达式
- 使用grep -E可以支持扩展正则表达式
- 使用egrep可以支持扩展正则表达式，与grep -E等价
