列表

无序列表

Input

- 列表内容
+ 列表内容
* 列表内容

Output

  • 列表内容
  • 列表内容
  • 列表内容

“.”跟内容之间要有空格

Input

2. 列表内容
3. 列表内容

Output

  1. 列表内容
  2. 列表内容
  3. 列表内容
