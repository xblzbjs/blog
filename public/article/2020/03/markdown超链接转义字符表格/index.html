超链接

有文字链接、索引链接、图片链接(地址)、自动链接

Input

# 文字链接
[xblzbjs的博客](https://xblzbjs.cn "xblzbjs的博客")
# 索引链接
[xblzbjs的博客][1]
[1]:https://xblzbjs.cn
# 自动链接
<https://xblzbjs.cn>

Output

  • xblzbjs的博客
  • 我的博客 博客
  • https://xblzbjs.cn
